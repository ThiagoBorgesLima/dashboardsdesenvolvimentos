Dashboard de Desenvolvimentos (Atualizável via CSV)

Objetivo: visualizar desenvolvimentos por máquina ao longo do tempo e identificar posições com maior necessidade de desenvolvimento.
Dica: salve este arquivo HTML e, ao abrir, carregue o CSV atualizado no seletor abaixo.

Entrada e filtros

Este dashboard tenta localizar automaticamente a linha de cabeçalho (onde aparece “Posição”). Se você mudar o layout do CSV, dá pra ajustar em 2 minutos no código.
Formato esperado: Data;Máquina;Código;Posição;Vestimenta;Fornecedor. A coluna Máquina pode vir como 1/3/6 (o dashboard converte para MP1/MP3/MP6).
Busca em Posição. Combina com os filtros de máquina e status.
